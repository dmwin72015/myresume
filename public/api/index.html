getElementsByClassName(className)H5

概述

根据提供的原始 HTML 标记字符串，动态创建由 jQuery 对象包装的 DOM 元素。同时设置一系列的属性、事件等。 你可以传递一个手写的 HTML 字符串，或者由某些模板引擎或插件创建的字符串，也可以是通过 AJAX 加载过来的字符串。但是在你创建 input

参数
className string

传入的class名称

className string

传入的class名称

示例
示例一

获取所有class名字为test的元素

javascript代码：
var aTest = document.getElementsByClassName('test');
示例二

获取所有class名字为test的元素

javascript代码：
var aTest = document.getElementsByClassName('test');
